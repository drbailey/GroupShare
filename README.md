GroupShare
==========

Learn python with friends
Lets do some python!
Some ideas of what to work on are...
	1.Project Euler
	2.Simple games
	3.Exercises and examples
	4.Anything you want other's feedback on
Feel free to add or change anything here.
	
	
## Resources:
